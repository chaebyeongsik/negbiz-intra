304 Beautiful Blocks for ContentBuilder.js

*** Installation ***

1. Copy the "minimalist" folder in "assets" folder.

2. On the page that uses the ContentBuilder.js, include the css on the head:

<link href="assets/minimalist/content.css" rel="stylesheet" type="text/css" />

3. Specify the snippet's file:

        $("#contentarea").contentbuilder({
            zoom: 0.85,
            snippetFile: 'assets/minimalist/snippets.html'
            });
